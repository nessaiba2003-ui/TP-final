Description

Ce projet capstone représente l'aboutissement du cours Hibernate & JPA. Il consiste à finaliser une application complète de **réservation de salles** en intégrant tous les concepts appris : modèle de données avancé, verrouillage optimiste, cache de second niveau, jeux de données réalistes, tests complets, migrations de base de données et optimisation des performances.

##  Objectifs du Projet

-  Finaliser l'application de réservation de salles
-  Créer des jeux de données réalistes
-  Développer des scénarios de test complets
-  Implémenter un script de migration de base de données
-  Réaliser un rapport de performance
-  Intégrer le verrouillage optimiste
-  Configurer le cache de second niveau
-  Optimiser les requêtes et résoudre le problème N+1

### Fonctionnalités Principales

1. **Gestion des Salles**
   - Création, modification, suppression de salles
   - Capacités et équipements des salles
   - Disponibilité en temps réel

2. **Système de Réservation**
   - Réservation de créneaux horaires
   - Gestion des conflits de réservation
   - Verrouillage optimiste pour éviter les doubles réservations
  
 3. **Gestion des Utilisateurs**
   - Profils utilisateurs
   - Permissions et rôles
   - Historique des réservations

4. **Recherche et Filtrage**
   - Recherche par capacité
   - Filtrage par équipements
   - Disponibilité par période

## 📚 Prérequis

- Java 11 ou supérieur
- Maven 
- Base de données PostgreSQL/MySQL/H2
